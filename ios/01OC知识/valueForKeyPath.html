引言:

以前一直没有用到过valueForKeyPath,一直以为和valueForKey差不多,可是今天一看才吓了一跳,功能好强大啊. valueForKeyPath是KVC方法,可以实现多级取值等功能.


### I. 最小值、最大值、平均值、求和 ____ valueForKeyPath可以获取数组中的最小值、最大值、平均值、求和。代码如下： 

NSArray *array = @[@10, @23, @43, @54, @7, @17, @5];

+ (NSString *)caculateArray:(NSArray *)array {

    CGFloat sum = [[array valueForKeyPath:@"@sum.floatValue"] floatValue]; // 求和
    CGFloat avg = [[array valueForKeyPath:@"@avg.floatValue"] floatValue]; // 平均值
    CGFloat max =[[array valueForKeyPath:@"@max.floatValue"] floatValue];  // 最大值
    CGFloat min =[[array valueForKeyPath:@"@min.floatValue"] floatValue];  // 最小值
    
    NSLog(@"%fn%fn%fn%f",sum,avg,max,min);
    return [NSString stringWithFormat:@"%f",sum];
}

### II. 获取数组中相同key的元素 ___

alueForKeyPath可以获取数组中相同key的元素:

NSArray *arr = @[@{@"city":@"beijing",@"person":@{@"name":@"zhangsan"}},@{@"city":@"chengdu"}];

NSArray *array = [arr valueForKeyPath:@"city"]; 
// 可以获取到city数组 @[@"beijing",@"chengdu"]

### III. 多级索引 ____

valueForKeyPath 可以使用. 来一层一层向下索引，当多个字典层级时，取子层级中的属性就非常简单了

NSDictionary *dict1 = @{@"dict1":@{@"dict2":@{@"name":@"lishi",@"info":@{@"age":@"12"}}}};

id res = [dict1 valueForKeyPath:@"dict1.dict2.name"];
NSLog(@"res = %@",res);
// 输出lishi

### IV. 删除重复的数据 ____

删除重复的数据 ： distinctUnionOfObjects.self

NSArray *array = @[@"qq", @"wechat", @"qq", @"msn", @"wechat"];
NSArray *deleltNewA = [array valueForKeyPath:@"@distinctUnionOfObjects.self"];
NSLog(@"deleltNewA = %@",deleltNewA);
// 输出( qq,  wechat,    msn)

### V. 嵌套使用 ____

嵌套使用（先删除对应值的重复数据再取值）

NSArray *array = @[@{@"name" : @"xiaoming",
                  @"code" : @1},
                 @{@"name": @"judy",
                  @"code" : @2},
                 @{@"name": @"judy",
                  @"code" : @3},
                 @{@"name": @"xiaohua",
                  @"code" : @4}];
                  
NSArray *deleltNewA = [array valueForKeyPath:@"@distinctUnionOfObjects.name"];

NSLog(@"deleltNewA = %@",deleltNewA);
// 输出(judy, xiaohua, xiaoming);
