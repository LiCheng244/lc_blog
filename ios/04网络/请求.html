iOS9.0+ http协议无法请求问题

引言:

iOS9引入了新特性App Transport Security (ATS)。

新特性要求App内访问的网络必须使用HTTPS协议。但是现在公司的项目使用的是HTTP协议，使用私有加密方式保证数据安全。现在也不能马上改成HTTPS协议传输。


# 一. 通过Info.plist文件解决

在Info.plist中添加 NSAppTransportSecurity类型为:Dictionary。 在 NSAppTransportSecurity 下添加 NSAllowsArbitraryLoads 类型为Boolean, 值设为YES。


# 二. 通过代码解决

将Info.plist文件 Source Code 打开, 输入下面代码:

     <key>NSAppTransportSecurity</key>
     <dict>
     <key>NSAllowsArbitraryLoads</key>
     <true/>
     </dict>
