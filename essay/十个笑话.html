十个笑话让你看懂人生


# 一. 关怀

一把坚实的大锁挂在大门上，一根铁杆费了九牛二虎之力，还是无法将它撬开。钥匙来了，他瘦小的身子钻进锁孔，只轻轻一转，大锁就“啪”一声打开了。

悟：

每个人的心，都像上了锁的大门，任你再粗的铁棒也撬不开。惟有关怀，才能把自己变成一把细腻的钥匙，进入别人的心中。


# 二. 内心态度

父子二人看到一辆豪华进口轿车。儿子不屑地对他的父亲说：“坐这种车的人，肚子里一定没有学问！”父亲则轻描淡写地回答：“说这种话的人，口袋里一定没有钱！”

悟：

你对事情的看法，是不是也反映出你内心真正态度？


# 三. 严以待己

晚饭后，母亲和女儿一块儿洗碗盘，父亲和儿子在客厅看电视。突然，厨房里传来打破盘子的响声，然后一片沉寂。儿子望着他父亲，说：“一定是妈妈打破的。”“你怎么知道？”“她没有骂人。”

悟：

我们习惯以不同标准来待人，以至于常严以待人，宽于待己。


# 四. 思考方式

有两个观光团到同一个地方旅游，路况很坏，到处都是坑洞。一位导游连声说，路面简直像麻子一样；而另一个导游却诗意盎然地对游客说：“我们现在走的正是赫赫有名的本地迷人酒窝大道。”

悟：

虽是同样的情况，辅以不同视角，就会产生不同态度。思想是奇妙的事，如何去想，决定权在你。


# 五. 鼓励

同样是小学三年级的学生，他们将来志愿同是当小丑。有的人斥之为：“胸无大志，孺子不可教也！”有的人则会说：“愿你把欢笑带给全世界！”

悟：

悟：身为长辈的我们，不但要求多于鼓励，更以狭窄界定成功。


# 六. 考虑对方

6、妻子正在厨房炒菜，丈夫在她旁边一直唠叨不停：“慢些！小心！火太大了！赶快把鱼翻过来！油放太多了！”妻子脱口而出：“我懂得怎样炒菜。”丈夫平静答道：“我只是要让你知道，我在开车时，你在旁边喋喋不休，我的感觉如何……”

悟：

学会体谅他人并不困难，只要你愿意认真站在对方的角度和立场看问题。


# 七. 全力以赴

一辆载满乘客的公共汽车沿着下坡路快速前进，有一个人在后面紧紧追赶这辆车子。一个乘客从车窗中伸出头来对追车子的人说：“老兄，算了，你追不上的！”“我必须追上它”，这人气喘吁吁地说，“我是这辆车的司机！”

悟：

有些人必须非常认真努力，因为不这样的话，后果就十分悲惨了！然而也正因为必须全力以赴，潜在的本能和不为人知的特质终将充分展现出来。


# 八. 事出必有因

甲：“新搬来的邻居好可恶，昨天晚上三更半夜跑来猛按我家的门铃。”乙：“的确可恶！你有没有马上报警？”甲：“没有。我当他们是疯子，继续吹我的小喇叭。”

悟：

事出必有因，如果能先看到自己的不是，答案就会不一样。


# 九. 误会

张三在山间小路开车，正当他悠哉欣赏美丽风景时，突然迎面开来的货车司机摇下窗户大喊一声：“猪！”张三越想越气，也摇下车窗大骂：“你才是猪！”刚骂完，他便迎头撞上一群过马路的猪。

悟：

不要错误地诠释别人的好意，那只会让自己吃亏，并且使别人受辱。


# 十. 权威

10、小男孩问爸爸：“是不是父亲总比儿子知道得多？”爸爸回答：“当然啦！”“电灯是谁发明的？”“爱迪生。”“那爱迪生的爸爸怎么没有发明电灯？”

悟：

权威往往只是一个经不起考验的空壳子，尤其在现今这个多元开放的时代。
